# Weather-Application

Developed a dynamic Weather Application using HTML, CSS, JavaScript, Java JSP, Servlets, and the OpenWeather API.
Implemented real-time weather data fetching and display functionality to provide users with up-to-date weather information.
Designed and optimized user-friendly frontend interfaces for enhanced user interaction and data visualization.
Leveraged Java Servlets for efficient server-side processing and JavaServer Pages (JSP) for dynamic content delivery.
Applied responsive design techniques to ensure compatibility and a seamless user experience across various devices.
